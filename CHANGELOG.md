# Changelog

All notable changes to the "Vue 3 + Tailwind + Bootstrap Snippets" extension will be documented in this file.

## [2.0.0] - 2024-01-XX

### Added
- **Vue 3 Composition API snippets** with full TypeScript support
  - `vbase` - Vue 3 SFC with Composition API (TypeScript)
  - `vbasejs` - Vue 3 SFC with Composition API (JavaScript)
  - `vref`, `vrefjs` - Reactive references
  - `vcomputed`, `vcomputedts` - Computed properties
  - `vwatch`, `vwatchopt`, `vwatcheffect` - Watch statements
  - `vprops`, `vpropsdef` - Props definitions
  - `vemits`, `vemitsjs` - Emits definitions
  - `vonmounted`, `vonunmounted`, `vonupdated` - Lifecycle hooks
  - `vreactive`, `vprovide`, `vinject` - Reactivity utilities
  - `vnexttick`, `vtemplateref` - Utility functions
  - `vfor`, `vforidx` - v-for directives
  - `vmodel`, `vmodelnum` - v-model directives
  - `vif`, `vshow` - Conditional rendering
  - `vclick`, `vsubmit` - Event handlers
  - `vrouterlink`, `vrouterview`, `vuserouter`, `vuseroute` - Vue Router
  - `vtransition`, `vteleport`, `vsuspense` - Vue 3 components
  - `vasynccomp` - Async components
  - `vpinia`, `vpinia-comp` - Pinia store snippets
  - `vnuxfetch`, `vnuxasync`, `vnuxhead` - Nuxt 3 composables

- **Tailwind CSS 3 & 4 snippets**
  - `twbtn`, `twbtn-primary`, `twbtn-secondary` - Button components
  - `twcard` - Card component
  - `twflex`, `twflex-col` - Flexbox utilities
  - `twgrid`, `twgrid-responsive` - Grid layouts
  - `twdark`, `twdark-class` - Dark mode support
  - `twcontainer` - Container component
  - `twtext-responsive` - Responsive text sizing
  - `twinput`, `twform` - Form components
  - `twnavbar`, `twmodal` - UI components
  - `twbadge`, `twalert`, `twspinner` - Utility components
  - `twsm`, `twmd`, `twlg`, `twxl`, `tw2xl` - Responsive breakpoints
  - `twhover`, `twfocus`, `twactive` - State variants
  - `twtransition`, `twshadow`, `twrounded` - Utility classes

- **Bootstrap 5 & 6 snippets**
  - `bsgrid`, `bsrow`, `bscol` - Grid system
  - `bsbtn`, `bsbtn-primary`, `bsbtn-secondary`, etc. - Button variants
  - `bsform`, `bsinput`, `bsselect`, `bscheckbox`, `bsradio` - Form components
  - `bsnavbar` - Navigation bar
  - `bsmodal` - Modal dialog
  - `bscard`, `bsalert`, `bsbadge` - UI components
  - `bsspinner`, `bsprogress` - Loading indicators
  - `bsdropdown`, `bstable` - Interactive components
  - `bscontainer`, `bscontainer-fluid` - Layout containers
  - `bsaccordion`, `bstoast`, `bstooltip`, `bspopover` - Advanced components

- **Modern HTML5 snippets**
  - Semantic HTML5 document structure
  - Accessible form elements with ARIA labels
  - Semantic elements (header, main, footer, article, section)
  - Modern HTML elements (details, dialog, picture, video, audio, svg)
  - Open Graph meta tags
  - Responsive image patterns

- **Pug template snippets**
  - Complete Pug syntax support
  - Semantic HTML structures in Pug
  - Mixins, includes, extends, blocks
  - Conditionals and loops
  - Accessible form patterns

- **ASP.NET Razor snippets**
  - Razor page and view templates
  - Layout and section definitions
  - Conditional statements and loops
  - Model binding patterns

- **Documentation**
  - Comprehensive README.md with snippet tables
  - Usage examples and code samples
  - Test cases and demo files
  - Compatibility information

### Changed
- Updated package.json with modern keywords and file associations
- Improved extension metadata and descriptions
- Enhanced snippet descriptions for better IntelliSense
